# Task List #

### Dependences ###

* Ruby 2.3.1
* Rails 4.2.6
* Postgresql

### Local Host
    127.0.0.1         local.task-list.com

## Setup
    bundle install
    rake db:create && rake db:migrate

## Run app
    rails s
    open http://local.task-list.com:3000

## Run tests
    rspec
